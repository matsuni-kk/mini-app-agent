# Mini App Build Questions

## 必須入力項目

1. **実装範囲**: 全機能を実装しますか？段階的に実装しますか？
2. **外部ライブラリ**: CDNで使用するライブラリはありますか？
3. **データ永続化**: ローカルストレージは使用しますか？

## 任意入力項目

4. **既存コード**: 既存のコードベースはありますか？
5. **特殊要件**: 特別な実装要件はありますか？

## 質問フロー

```
1. requirements.md / design.md 読み込み
   ↓
2. 必須項目（1-3）を確認
   ↓
3. 任意項目（4-5）を該当あれば確認
   ↓
4. 実装開始
```

## 回答テンプレート

```markdown
## 実装方針
- 実装範囲:
- 外部ライブラリ:
- データ永続化:

## その他
- 既存コード:
- 特殊要件:
```
