# Mini App Design Questions

## 必須入力項目

1. **画面数**: 何画面構成ですか？（単一画面/複数画面）
2. **主要コンポーネント**: どのようなUI部品が必要ですか？
3. **レスポンシブ**: どのデバイスに対応しますか？

## 任意入力項目

4. **カラー**: 好みの色やテーマは？
5. **参考UI**: 参考にしたいUIはありますか？
6. **アイコン**: アイコンは使用しますか？（Font Awesome等）
7. **アニメーション**: アニメーションは必要ですか？

## 質問フロー

```
1. requirements.md読み込み
   ↓
2. 必須項目（1-3）を確認
   ↓
3. 任意項目（4-7）を該当あれば確認
   ↓
4. 設計確認・承認依頼
```
