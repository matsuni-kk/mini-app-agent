# Mini App Design 評価指標

> 参照: [design_system_principles.md](../assets/design_system_principles.md) - ADS統合評価システム

## 評価思想

**品質 ≠ 印象**：主観的フィードバックを客観的スコアで制御する。

```
総合品質評価 = 情報密度 × コンテキスト × 美的評価 × 発見可能性
```

---

## 必須チェック項目

### 構造チェック（Pass/Fail）

| 項目 | 基準 | 重要度 |
|------|------|--------|
| テンプレート準拠 | 必須セクション（画面一覧/画面設計/コンポーネント/ビジュアル/技術設計）が存在する | Critical |
| 要件整合性 | requirements.mdの全機能要件に対応する画面が設計されている | Critical |
| ファイル構成定義 | 実装に必要なファイル構成が明記されている | Critical |
| 未記載項目の明示 | 不明項目は「未記載」「不明」と明記されている | High |
| レスポンシブ方針 | 対応デバイスと各ブレークポイントが定義されている | High |
| デザインシステム準拠 | design_system_principles.mdの原則に従っている | High |

### 内容チェック（スコアリング）

| 観点 | 評価基準 | 配点 |
|------|----------|------|
| 完全性 | 全機能要件に対応する画面・コンポーネントが設計されている | 25 |
| 明確性 | ワイヤーフレームが具体的でインタラクションが明示されている | 20 |
| 実装可能性 | HTML/CSS/JSで実装可能な設計になっている | 20 |
| 一貫性 | カラー・タイポグラフィ・コンポーネントの統一感 | 15 |
| ADS品質指標 | 強調・美的評価・情報密度・コンテキストの適切さ | 20 |

## ADS品質指標（詳細）

### 強調システム（Emphasis System）

CTAの優先度が適切か評価。

```
総合強調度 = コントラスト × サイズ × 余白 × 面積 × 動き × 太さ
```

| チェック項目 | 基準 |
|-------------|------|
| 主CTAの強調度 | 他要素より明確に目立つ（コントラスト4.5:1以上、サイズ1.5倍以上） |
| 視覚階層 | 重要度順に強調度が設計されている |
| 余白の効果 | 重要要素は十分な余白で視線誘導 |

### 美的評価（Aesthetic Evaluation）

```
美的評価 = ブランド適合度 × 色彩調和 × 統一性
```

| チェック項目 | 基準 |
|-------------|------|
| ブランド適合度 | カラー・トーンがブランドイメージと一致 |
| 色彩調和 | 補色/類似色/トライアドの整合性 |
| 統一性 | パターン・余白・形状が一貫している |

### 情報密度（Information Density）

```
情報密度 = 有効情報量 / 視覚面積
```

| チェック項目 | 基準 |
|-------------|------|
| 適正密度 | 1画面で主要タスクが完結する範囲 |
| 認知負荷 | 過密で離脱を招いていない |
| スクロール効率 | 過疎でスクロール量が増えすぎていない |

### コンテキスト適応

| チェック項目 | 基準 |
|-------------|------|
| デバイス対応 | タッチ/マウス両対応（CTA 44×44px以上） |
| レスポンシブ | ブレークポイント毎にレイアウト最適化 |
| アクセシビリティ | キーボード操作・フォーカス可視化対応 |

---

## 認知科学チェック

| 法則 | チェック項目 |
|------|-------------|
| ヒックの法則 | 主要導線の選択肢が3〜7個に収まっているか |
| フィッツの法則 | 主CTAが44×44px以上、親指が届く位置か |
| ミラーの法則 | フォームはステップ分割、一覧は意味のある塊か |
| ヤコブの法則 | ナビゲーションは慣れた型を優先しているか |

---

### 許容例外

- アニメーション仕様が未定義（要件にない場合）
- アイコンライブラリが未指定（使用しない場合）

## 採点基準

- **Pass**: 全Criticalチェック項目がPass かつ スコア80点以上
- **Conditional Pass**: 全Criticalチェック項目がPass かつ スコア60-79点
- **Fail**: CriticalチェックにFailあり または スコア60点未満
