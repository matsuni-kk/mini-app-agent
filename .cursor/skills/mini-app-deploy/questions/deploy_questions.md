# Mini App Deploy Questions

## 必須入力項目

1. **リポジトリ名**: リポジトリ名は何にしますか？（デフォルト: アプリ名）
2. **GitHubユーザー名**: デプロイ先のGitHubアカウント名は？
3. **Vercelプロジェクト名**: Vercelのプロジェクト名は？（デフォルト: リポジトリ名）

## 任意入力項目

4. **カスタムドメイン**: カスタムドメインを使用しますか？
5. **説明文**: リポジトリの説明文は？

## 質問フロー

```
1. test_report.md確認（Passであること）
   ↓
2. 必須項目（1-3）を確認
   ↓
3. 任意項目（4-5）を該当あれば確認
   ↓
4. デプロイ実行
```

## 回答テンプレート

```markdown
## デプロイ設定
- リポジトリ名:
- GitHubユーザー名:
- Vercelプロジェクト名:
- 公開設定: Private（プライベートリポジトリからデプロイ）

## オプション
- カスタムドメイン:
- 説明文:
```

## 公開URL計算

```
https://{{Vercelプロジェクト名}}.vercel.app
```

## 備考

- リポジトリは常にPrivate（プライベート）で作成
- Vercelは無料プランでもプライベートリポジトリからのデプロイをサポート
- サイト自体は公開URLでアクセス可能（ソースコードは非公開）
