# Mini App Requirements Questions

## 必須入力項目

1. **アプリ名**: このアプリの名前は？
2. **目的**: このアプリで何を実現したいですか？
3. **対象ユーザー**: 誰が使いますか？
4. **必須機能**: 絶対に必要な機能は？（最低1つ）

## 任意入力項目

5. **追加機能**: あると嬉しい機能は？
6. **レスポンシブ**: スマホ対応は必要ですか？（デフォルト: はい）
7. **参考アプリ**: 参考にしたいデザインやアプリはありますか？
8. **制約条件**: 制約や注意事項はありますか？
9. **色の好み**: 色やテーマの好みはありますか？
10. **成功基準**: 成功の基準は何ですか？

## 質問フロー

```
1. 必須項目（1-4）を確認
   ↓
2. 任意項目（5-10）を該当あれば確認
   ↓
3. 要件確認・承認依頼
```
