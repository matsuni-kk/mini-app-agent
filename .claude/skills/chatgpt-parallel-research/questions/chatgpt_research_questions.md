# ChatGPT Parallel Research Questions

## 必須入力項目

### 1. 検索目的
- **何を調べたいですか？**
  - 例: 技術調査、市場調査、競合分析、アイデア出し、問題解決

### 2. 検索テーマ・キーワード
- **調査対象のテーマ・キーワードは？**
  - 例: 「React Server Components」「AIエージェントの最新動向」

### 3. 期待成果
- **最終的にどんな成果物が欲しいですか？**
  - 例: 比較表、選択肢リスト、技術まとめ、アイデアリスト、課題と解決策

### 4. 検索クエリ設計（3個以上必須）
- **どのような角度から調べますか？**
  - 異なる視点・切り口で最低3つのクエリを設計
  - 例:
    1. 「〇〇の基本概念と仕組みを解説して」
    2. 「〇〇のメリット・デメリットを整理して」
    3. 「〇〇の具体的なユースケースを5つ挙げて」
    4. 「〇〇と△△の違いを比較して」

## 難易度判定

### 5. 問題の難易度
- **どの程度の難易度ですか？**

| レベル | 説明 | 推奨設定 |
|--------|------|----------|
| 簡単 | 一般的な情報収集、簡単なブレスト | gpt-4o |
| 普通 | 比較分析、中程度のブレスト | gpt-4o |
| やや難 | 技術的な深掘り、複雑な分析 | o3 + standard |
| 難問 | 専門的な分析、高精度が必要 | o3 + heavy |
| 最難関 | 最大限の深い思考が必要 | o3 + extended |

## 任意入力項目

### 6. 制約条件
- **制約や前提条件はありますか？**
  - 例: 日本語のみ、最新情報のみ、特定のドメイン限定

### 7. 参照資料
- **参考にすべき既存資料はありますか？**
  - 例: 過去の調査結果、公式ドキュメントURL

### 8. 優先順位
- **検索の優先順位は？**
  - 例: 正確性重視、網羅性重視、速度重視

## クエリ設計のヒント

### ウェブ検索系
- 「〇〇について最新情報を検索して」
- 「△△の公式ドキュメントの内容を教えて」
- 「〇〇の2024年の動向を調べて」

### ブレスト系
- 「〇〇のアイデアを10個出して」
- 「△△の課題と解決策をブレストして」
- 「〇〇の新しいアプローチを提案して」

### 分析系
- 「AとBの違いを詳しく比較して」
- 「〇〇の選択肢のメリデメを整理して」
- 「△△のアーキテクチャを分析して」

### 深掘り系
- 「〇〇の技術的な仕組みを詳しく解説して」
- 「△△の設計パターンを分析して」
- 「〇〇のベストプラクティスを教えて」
