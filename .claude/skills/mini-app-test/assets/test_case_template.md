# テストケーステンプレート

## テストケース形式

```markdown
### TC-XXX: {{テストケース名}}

**前提条件**
- {{前提1}}
- {{前提2}}

**テスト手順**
1. {{手順1}}
2. {{手順2}}
3. {{手順3}}

**期待結果**
- {{期待結果}}

**実際の結果**
- {{実際の結果}}

**判定**: Pass / Fail

**備考**
- {{備考}}
```

## テストカテゴリ別テンプレート

### 機能テスト

```markdown
### TC-F001: {{機能名}}の正常動作

**対応要件**: {{M1/S1等}}

**前提条件**
- アプリが正常に読み込まれている

**テスト手順**
1. {{操作1}}
2. {{操作2}}

**期待結果**
- {{期待する動作}}

**判定**: Pass / Fail
```

### 境界値テスト

```markdown
### TC-B001: {{項目名}}の境界値

**テストデータ**
| 入力値 | 期待結果 | 実際の結果 | 判定 |
|--------|----------|------------|------|
| 最小値 | {{期待}} | {{実際}} | Pass/Fail |
| 最小値-1 | {{期待}} | {{実際}} | Pass/Fail |
| 最大値 | {{期待}} | {{実際}} | Pass/Fail |
| 最大値+1 | {{期待}} | {{実際}} | Pass/Fail |
```

### 異常系テスト

```markdown
### TC-E001: {{異常ケース名}}

**前提条件**
- {{異常状態の設定}}

**テスト手順**
1. {{異常を引き起こす操作}}

**期待結果**
- エラーメッセージが表示される
- アプリがクラッシュしない
- {{その他の期待}}

**判定**: Pass / Fail
```

### レスポンシブテスト

```markdown
### TC-R001: {{画面名}}のレスポンシブ表示

**テスト環境**
| デバイス | 解像度 | ブラウザ |
|----------|--------|----------|
| Desktop | 1920x1080 | Chrome |
| Tablet | 768x1024 | Safari |
| Mobile | 375x667 | Chrome |

**確認項目**
| 項目 | Desktop | Tablet | Mobile |
|------|---------|--------|--------|
| レイアウト崩れ | OK/NG | OK/NG | OK/NG |
| 文字切れ | OK/NG | OK/NG | OK/NG |
| タップ領域 | - | OK/NG | OK/NG |
```

## テストケースID規則

| カテゴリ | プレフィックス | 例 |
|----------|----------------|-----|
| 機能テスト | TC-F | TC-F001 |
| 境界値テスト | TC-B | TC-B001 |
| 異常系テスト | TC-E | TC-E001 |
| レスポンシブ | TC-R | TC-R001 |
| アクセシビリティ | TC-A | TC-A001 |
| パフォーマンス | TC-P | TC-P001 |
