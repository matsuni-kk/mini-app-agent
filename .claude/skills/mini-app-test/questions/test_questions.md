# Mini App Test Questions

## 必須入力項目

1. **テスト範囲**: どの機能をテストしますか？（全機能/Must機能のみ/特定機能）
2. **テスト環境**: どのブラウザ・デバイスでテストしますか？
3. **重点項目**: 特に重点的にテストすべき項目はありますか？

## 任意入力項目

4. **既知の問題**: 既に把握している不具合はありますか？
5. **スキップ項目**: テストをスキップする項目はありますか？
6. **自動テスト**: 自動テストツールを使用しますか？

## 質問フロー

```
1. requirements.md / design.md / ソースコード読み込み
   ↓
2. 必須項目（1-3）を確認
   ↓
3. 任意項目（4-6）を該当あれば確認
   ↓
4. テスト実施
```
