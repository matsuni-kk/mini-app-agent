# Mini App Deploy Questions

## 必須入力項目

1. **リポジトリ名**: リポジトリ名は何にしますか？（デフォルト: アプリ名）
2. **公開設定**: Public（公開）/ Private（非公開）どちらにしますか？
3. **GitHubユーザー名**: デプロイ先のGitHubアカウント名は？

## 任意入力項目

4. **カスタムドメイン**: カスタムドメインを使用しますか？
5. **説明文**: リポジトリの説明文は？

## 質問フロー

```
1. test_report.md確認（Passであること）
   ↓
2. 必須項目（1-3）を確認
   ↓
3. 任意項目（4-5）を該当あれば確認
   ↓
4. デプロイ実行
```

## 回答テンプレート

```markdown
## デプロイ設定
- リポジトリ名:
- 公開設定: Public / Private
- GitHubユーザー名:

## オプション
- カスタムドメイン:
- 説明文:
```

## 公開URL計算

```
https://{{GitHubユーザー名}}.github.io/{{リポジトリ名}}/
```
